тех задание:
Технологии: React, Js, HTML, CSS

Требуется реализовать страницу, на которой будет отображена схема зала (см. скриншот ниже), файл SVG доступен по ссылке
На странице реализованы две кнопки "+" и "-", которые могут масштабировать схему зала (приближать и удалять)
Схему зала можно "таскать" зажав мышкой.
Схема SVG должна загружаться из внешнего файла, то есть эмулироваться загрузка схемы зала из API.
Места на схеме зала могут быть раскрашены произвольным образом. При клике на место, оно выделяется.
К выполненному заданию необходимо приложить краткое описание примененных технологий и методов разработки.

Выполненное задание можно прислать в виде ссылки на опубликованную веб-страницу либо архива с файлами.

В качестве бонусного задания можно реализовать возможность масштабировать схему на мобильном устройстве жестами двух пальцев ("стягивание" и "растягивание").
